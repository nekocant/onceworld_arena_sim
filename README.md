# ⚔️ OnceWorld アリーナ勝敗予想シミュレーター

## 概要
OnceWorld アリーナ勝敗予想シミュレーターは、Streamlitを利用して構築された仮想モンスターのオートバトラー・シミュレーションアプリです。
プレイヤーは提供されたモンスターデータを元にチームA、B、Cを結成し、各チーム間での戦闘の行方や勝敗を予想・シミュレートすることができます。

## ディレクトリ構成と各ファイルの役割

現在のプロジェクトディレクトリは、本番環境で実際に使用されている以下のファイル群で構成されています。不要なファイルは存在せず、すべてがアプリの動作に必要なファイルです。

- **`app_arena.py`**
  - **役割**: メインのUIおよびアプリケーションエンドポイント（Streamlitアプリ）。
  - **詳細**: チームごとの選出画面（ランダム/手動）、各モンスターのステータス比較テーブル（進捗バー付きUI）、画面のレスポンシブ対応などを管理します。戦闘ログの表示やマップ上でのビジュアル観戦機能の呼び出しも行います。

- **`battle_logic.py`**
  - **役割**: 戦闘のバックエンド・シミュレーションロジック。
  - **詳細**: `Monster`クラスおよび`Field`クラスを定義。ステータスのレベル補正計算、移動処理（ターゲットロック・接近）、攻撃処理（多段ヒット、命中・回避、クリティカル、属性相性など）からなる緻密な戦闘計算を担います。

- **`monsters.csv`**
  - **役割**: モンスターの基本ステータスマスターデータ。
  - **詳細**: モンスターの「NO.」、名前、属性（火・水・木・光・闇）、物魔タイプ（物理・魔法）、基礎攻撃力や耐久値などが定義されています。

- **`attack_range.json`**
  - **役割**: 特定モンスター固有の攻撃範囲のカスタム定義。
  - **詳細**: `battle_logic.py`側で読み込まれ、一部のモンスターに対して基礎値からの上書き適用（独自の攻撃レンジの付与）を行います。

- **`kana_mapping.json`**
  - **役割**: 検索機能用のよみがなマッピングデータ。
  - **詳細**: `app_arena.py`で手動選出を行う際、漢字名を持つモンスターを「ひらがな」や「カタカナ」でスムーズにあいまい検索できるようにするための辞書データです。

- **`requirements.txt`**
  - **役割**: 必要なPythonパッケージ定義。
  - **詳細**: `streamlit`, `pandas` の依存関係が記載されています。

- **`.streamlit/`**
  - **役割**: Streamlitの設定ディレクトリ。
  - **詳細**: サーバーのポートやテーマカラーなどの設定ファイル(`config.toml`等)が配置されています。

## 特徴
- **詳細な戦闘ロジック**: SPDに基づく攻撃回数の多段ヒット処理や、Luck値に基づく回避・クリティカル判定など、高度な戦闘アルゴリズムを実装しています。
- **リッチなUI**: 全モンスターの強さを一目で比較できる偏差ステータスバーや、特定チームの色分けを行っています。

## 💡 こだわり機能（入力・検索システム）
本アプリでは、ユーザーが膨大なモンスターの中から目的の1体を見つけやすくするため、入力体験と検索ロジックに深くこだわっています。

- **自動ひらがな変換（WanaKana.js 連携）**
  - 数値入力を触った後にIME（日本語入力）がオフになるブラウザの制約を回避するため、検索ボックス上で**ローマ字入力をリアルタイムでひらがなに変換**するインジェクションを実装しています。
- **英単語・大文字入力の保護**
  - 「BOX」や「EXP」などの大文字英語は変換対象から除外されるため、日本語と英語が混在するモンスター名もストレスなく検索可能です。
- **高度なあいまい検索**
  - `kana_mapping.json` と連携し、漢字表記のモンスター（「白竜」「原始蝶」など）をひらがな・カタカナ・あるいは一部の読み方（「りゅう」等）でヒットさせる独自の検索ロジックを搭載しています。
- **UXの細かな調整**
  - **フォーカス時の全選択**: レベル入力や検索窓をクリックした際、既存のテキストが自動で全選択されるため、スムーズな上書きが可能です。
  - **読み取り専用セレクトボックス**: キーボード入力による誤操作を防ぎ、マウス選択に特化したスマートなUIを実現しています。

### Webサイト（公開URL）
ブラウザから直接アクセスしてシミュレーションを試すことができます：
[OnceWorld アリーナ勝敗予想シミュレーター](https://onceworldarenasim-dpw2irvkrb2nksbvynbaws.streamlit.app/)

### ローカルでの実行方法
開発環境で実行する場合は、以下のコマンドを使用してください：
```bash
pip install -r requirements.txt
python -m streamlit run app_arena.py
```
